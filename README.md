# Personal Scrolls (Next.js + TypeScript)

This is a minimal, production-ready Personal Scrolls built with Next.js, TypeScript and Tailwind CSS.

## Features
- Next.js with TypeScript
- Tailwind CSS
- Markdown posts in `/posts` (frontmatter parsed with `gray-matter`)
- Rendered with `react-markdown` + `remark-gfm`
- Dark / Light theme toggle
- Search, tags, and pagination
- RSS feed and `sitemap.xml` generated at build time
- Simple contact API that logs submissions to the server console
